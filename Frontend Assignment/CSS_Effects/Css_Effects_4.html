In this example, we have a container div with a width of 300px and a height of 200px, and a border of 1px solid black. We have also set overflow: scroll on the container to add scroll functionality. Inside the container, we have a content div with a width and height that are larger than the container. This will cause the content to overflow the container. When you open the HTML file in your web browser, you should see the container with scrollbars that allow you to scroll the content inside.
